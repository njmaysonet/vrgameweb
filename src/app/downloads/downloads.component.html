System Requirements

In order to run the game your system must meet the following requirements:

    Minimum System Requirements
  • OS: Windows 7 SP1, Windows 8.1, or Windows 10.
  • CPU: Intel i5-4590, AMD FX8350 equivalent or better.
  • RAM: 4GB or more.
  • Headset: HTC Vive.

Requires Steam© and SteamVR© installed on your system.
